# 0\. Rezumat executiv

Neural Network Development Dashboard (2N2D) este o platformă web inovatoare, menită să eficientizeze și să democratizeze procesul de dezvoltare a modelelor de Machine Learning. Platforma oferă un set complet de unelte integrate care se adresează atât începătorilor, cât și experților, permițând vizualizarea interactivă a arhitecturilor ONNX, preprocesarea avansată a datelor CSV și, cel mai important, optimizarea automată a rețelelor neuronale prin trei strategii distincte: Brute Force, algoritmi genetici (DEAP) și neuro-evoluție (NEAT).

Nucleul tehnic al proiectului constă în capacitatea originală de a analiza dinamic orice fișier ONNX, de a-i infera arhitectura și de a-l reconstrui ca un model Pytorch complet funcțional, gata de re-antrenare și optimizare. Construit pe o arhitectură de microservicii, proiectul integrează un frontend modern (Next.js, React, TypeScript) cu un backend performant (Python, FastAPI, Pytorch), utilizând servicii cloud pentru autentificare (Firebase), stocare (Cloudflare R2) și baze de date (PostgreSQL). Prin integrarea inteligentă a acestor tehnologii într-o arhitectură originală și o interfață intuitivă, 2N2D transformă un flux de lucru complex într-un proces accesibil și eficient.
